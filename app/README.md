## Setup
- Clone

```
git clone git@gitlab.asoft-python.com:huy.pham/nextjs-training.git
```

- Checkout to branch

```
git checkout feature/nextjs-13-practice
```

- Install package

```
pnpm install
```

- Run

```
pnpm dev
```
